Bem-vindo ao Docs!

A Ionic Component Documentation apresenta vários componentes úteis que são incluídos fora da caixa com o Ionic.

O que é Ionic?

O Ionic Framework é um SDK open source que permite aos desenvolvedores criar aplicativos móveis de alta qualidade com tecnologias Web, como HTML, CSS e JavaScript.

O que é Ionic Cloud?

O Ionic Cloud é uma plataforma em nuvem para gerenciar e dimensionar aplicativos Ionic com serviços integrados, como notificações por push, compilações nativas, autenticação de usuário e atualização em tempo real.
